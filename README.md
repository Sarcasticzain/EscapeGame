# 🚔 Escape Game — Outrun the Police

> A browser-based escape game built with pure HTML, CSS & JavaScript. You're being chased — think fast, move smart, and break free before it's too late.

🎮 **[Play Now](https://escapegame.ct.ws/)**

---

## 🕹️ About the Game

**Escape Game** is a thrilling browser game where you're on the run from the police. Navigate through obstacles, outsmart your pursuers, and find your way to freedom — all without installing a single thing.

No frameworks. No libraries. Just raw HTML, CSS, and JavaScript.

---

## ✨ Features

- 🚨 Police chase gameplay — stay one step ahead
- 🌐 Runs entirely in the browser — no downloads needed
- ⚡ Built with vanilla HTML, CSS & JavaScript
- 📱 Playable on desktop and mobile browsers
- 🎯 Fast-paced, reflex-testing challenges

## 🎮 How to Play

1. Open the game in your browser
2. Use **arrow keys** or **WASD** to move
3. Avoid the police and escape through the level
4. Survive as long as possible — beat your high score!
